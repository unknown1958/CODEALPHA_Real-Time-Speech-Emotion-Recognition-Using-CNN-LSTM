# Real-Time Speech Emotion Recognition Using CNN-LSTM

This project implements a Real-Time Speech Emotion Recognition (SER) system using a hybrid CNN-LSTM deep learning architecture. The system analyzes speech audio, extracts Mel Spectrogram features, and classifies emotions such as Happy, Sad, Angry, Fear, and Neutral.

The project combines audio signal processing and deep learning to capture both spatial and temporal speech characteristics. CNN layers are used for feature extraction from Mel Spectrograms, while LSTM layers learn temporal dependencies in speech signals. The trained model can be extended for real-time microphone-based emotion detection.

## Features

* Speech Emotion Recognition using Deep Learning
* Mel Spectrogram Feature Extraction
* CNN-LSTM Hybrid Architecture
* RAVDESS Dataset Support
* Real-Time Audio Prediction
* Performance Evaluation using Accuracy, Precision, Recall, F1-Score, and Confusion Matrix
